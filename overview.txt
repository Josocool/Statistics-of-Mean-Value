so we using various data sections of Statistics to indicate general characteristics of the data, so consisting of 3 things :

1. Finding the median value from available data ( Measures of central Tendency )
    
    there're 3 numbers to Find the middle value :
    
    + Mean value is Finding to Average of the data.
    + Median value is standard or middle value of the data.
    + Mode value is Moderate value of the data.

    + Variance ຫຼື ຄວາມແປປວນ ຄື : ການຫາຕົວຊີ້ວັດທີ່ບວກວ່າ ຄ່າຂໍ້ມູນຂອງແຕ່ລະຊຸດກະຈາຍຕົວອອກຈາກ ຄ່າສະເລ່ຍ (Mean) ຫຼາຍນ້ອຍສ່ຳໃດ ?
    + Standard Deviation ຫຼື ຄ່າບຽງເບນຈາກຄ່າມາດຕະຖານ ຄື : ຄ່າທີ່ໃຊ້ວັດການກະຈາຍຂອງຂໍ້ມູນ ແບບດຽວກັບ Variance ແຕ່ວ່າແບບ Standard Deviation ຈະມີຄ່າທີໜ່ວຍດຽວກັບຂໍ້ມູນຕົ້ນສະບັບ
